# indevlabs-assessment

## Downloading

```
git clone https://github.com/Azimkhan93/nodejs2024Q1-service.git
```

## Route to project folder
```
cd indevlabs-assessment
```

## Switching to the branch
```
git checkout dev
```
## Installing NPM modules

```
npm install
```

## Running the function

```
node assessment.js
```

## Testing

```
npm run test 
```
